# LOTTE ROAMING — 제품 기획 개요

> 외부 설명용 요약. 현재 구현된 데모(MVP) 기능을 기준으로 작성.
> 라이브: https://milesclub.github.io/lotte-roaming/

---

## 1. 한 줄 소개

**LOTTE ROAMING**은 여행 데이터(eSIM·USIM)를 파는 데서 그치지 않고, **출발 전 → 여행 중 → 귀국까지**를
하나로 잇는 롯데·L.POINT 제휴 여행 데이터 서비스입니다. 한국에 오는 외국인 여행자(인바운드)와
해외로 나가는 여행자(아웃바운드)를 모두 지원합니다.

## 2. 문제의식 & 기회

- 여행 eSIM은 **구매 즉시 관계가 끝나는** 일회성 상품으로 소비되어 재방문·교차판매가 약함.
- 반면 데이터 회선은 여행 내내 켜져 있는 **상시 연결 채널** → 적절한 순간에 혜택·정보를 전달할 최적의 접점.
- 롯데는 호텔·면세·백화점·관광 등 **국내외 오프라인 자산**이 강함 → 데이터 + 멤버십(L.POINT)을 묶으면
  "통신 상품"을 **여행 동반자(컴패니언)**로 확장할 수 있음.

## 3. 타깃 사용자

| 구분 | 사용자 | 핵심 니즈 |
|---|---|---|
| **인바운드** | 한국에 오는 외국인 여행자 | 도착 즉시 연결, 쉬운 개통, 한국 쇼핑·관광·면세 혜택 |
| **아웃바운드** | 해외로 가는 (주로 한국인) 여행자 | 여행지별 로밍, 현지 정보, 귀국 후 L.POINT 적립 |

홈 상단의 **"어디로 가시나요?"** 토글로 두 방향을 즉시 전환하며, 이후 모든 콘텐츠(상품·혜택·이미지·리뷰)가
방향에 맞게 바뀝니다.

## 4. 핵심 가치 제안

1. **몇 분 만에 개통, 약정 없음** — eSIM QR 즉시 설치 / USIM 공항 수령·배송.
2. **여행 내내 함께** — 개통 후 사용량·현지 정보·회화·혜택을 한 앱에서.
3. **롯데·L.POINT 혜택 연동** — 쇼핑·관광·숙박·면세, 국내외 롯데 자산 + 귀국 후 적립.
4. **언어 장벽 제거** — 4개 언어 UI + 리뷰 자동 번역 + 현지 회화 번역기.

## 5. 핵심 기능

### A. 구매 — "맞춤 구성(Builder)" 중심
- **홈 히어로의 플랜 빌더**: ① 여행지 선택(검색 지원) → ② 유형(데일리/종량제) → ③ 용량/기간 → ④ 예상 금액 → 상세 진입.
  - 데일리: 일일 데이터(500MB·1·2·3GB·무제한) + 이용 일수.
  - 종량제: 총 용량(3·5·10·20·30·50GB) + 유효기간(15/30일).
  - 유형별 설명 문구로 차이를 즉시 이해(매일 갱신 vs 기간 내 총량).
- **권역별 상품**: 한국·일본·아시아·동남아·유럽·북미·오세아니아·글로벌.
- **상품 상세**: SIM(eSIM/USIM)·연결망·옵션 선택, 포함 혜택, 유의사항, 모바일 고정 신청바. 옵션을 바꾸면
  **가격이 실시간으로** 반영.

### B. 계정 & 멤버십 — 구매와 분리
- 소셜(카카오·LINE·구글·애플·위챗) + 이메일 로그인/가입, 가입 시 **L.POINT 멤버십** 동반 생성.
- 이용약관 동의는 **가입 시 1회**(기존 회원은 매번 X). **로그인만** 따로도 가능(구매와 비결합).

### C. 신청 → 개통
- 신청자 정보 → 확인 → 완료(목업 결제 + eSIM 프로비저닝 QR / USIM 수령코드).
- **MY(신청 내역)** 목록·상세 + 상태 배지, **개통 안내**(기기별 단계 가이드).

### D. 개통 후 동반 경험 — "여행" 탭 (탐색기 + 탭 구성)
여행지를 고르면(이용 중이면 자동 선택) **혜택 / 정보 / 회화 / 추천** 탭으로 정리:
- **데이터 사용량 대시보드** — 잔여 데이터를 **액체가 차오르는 게이지**로 시각화, 남은 일수, **소진 임박 경고**, **충전 CTA**.
- **여행 실용 정보 팩** — 국가별 긴급번호(경찰·구급), 전압·플러그, **환율 계산기**, **현지 시각·시차**.
- **여행 회화 번역기** — 상황별 필수 문장(현지어 자동 번역) + 직접 입력 번역 + **음성 읽기(TTS)**.
- **관광 추천 가이드** — (한국 등) 지역별 롯데 숙박·관광·쇼핑 추천.

### E. 롯데 · L.POINT 혜택 — 권역 인지형
- 선택 여행지에 맞춰 **세계 각국 롯데 자산**을 노출(예: 뉴욕 롯데 팰리스, 롯데호텔 괌/하노이/시애틀, 롯데면세점 등).
- 모바일은 **카드 스택(스와이프)**, 데스크탑은 **그리드**로 동일 콘텐츠를 폭에 맞게 표현.
- 혜택 문구는 확정 전이라 **"제공 예정"** 안전 표현으로 통일.

### F. 신뢰 — 리뷰 & 자동 번역
- 사진·별점 기반 여행자 후기, **사용자 직접 작성** 기능(별점·내용·이미지).
- **타국어로 쓰인 리뷰를 설정 언어로 자동 번역**하고 **"번역됨" 배지 + 원문 보기** 토글 제공. 작성 언어를
  태깅해 다른 언어 사용자에게도 번역 노출.

### G. 알림 / 리마인더
- 최신 신청의 여행 날짜 기반 **출발 D-day / 여행 중** 홈 배너.
- 데이터 소진 임박 시 사용량 카드 내 경고. (실제 푸시 연동은 `push.ts` 시드로 준비.)

## 6. 대표 사용자 여정

```
[홈] 방향 선택 → 빌더로 여행지·유형·용량 구성 → 예상가 확인
  → [상세] 옵션·혜택·유의사항 확인 → 신청
  → (로그인/가입 · L.POINT 생성) → 정보 입력 → 확인 → 완료(QR/수령)
  → [개통] 도착 후 로밍 ON
  → [여행 탭] 사용량·현지정보·회화·혜택 활용 → (필요 시) 데이터 충전
  → 후기 작성 → 귀국 후 L.POINT 적립(예정)
```

## 7. 차별화 포인트

- **"파는 상품"이 아니라 "여행 동반자"** — 개통 후 경험(사용량·정보·회화·혜택)이 제품의 절반.
- **양방향(인바운드/아웃바운드)** 단일 서비스 — 외국인의 방한과 내국인의 출국을 한 흐름으로.
- **롯데 오프라인 자산과의 연결** — 통신을 미끼로 쇼핑·관광·면세·숙박까지 확장.
- **언어 장벽 제거** — UI 다국어 + 리뷰/회화 실시간 번역.

## 8. 디자인 원칙

- **L.POINT 디자인 토큰만 사용** — 브랜드 레드(#fb1822), 크림→라일락 그라데이션 등 정의된 토큰 한정(임의 색 금지).
- **플랫 디자인** — 드롭섀도우 없이 헤어라인·소프트 배경으로 위계 표현.
- **모바일 퍼스트** — 하단 네비, 모바일에선 헤더가 내비게이션 겸용, 네이티브 래핑 가능한 PWA 구조.
- **인터랙티브 컴포넌트** — 카드 스택·캐러셀·등장 애니메이션·액체 게이지 등으로 생동감.

## 9. 다국어 / 현지화

- **EN · KO · ZH · JA** 지원. 영어가 기준이며 각 언어가 키를 덮어쓰고 미번역은 영어로 폴백.
- 데이터(여행지·혜택·리뷰)는 구조 + 영문을 기본으로 두고 언어별 카탈로그가 보강.
- 핵심 흐름(구매·신청·여행 탭 라벨·번역 배지)은 4개 언어 정비, 일부 신규 라벨은 영어 폴백.

## 10. 기술 개요 & 확장성

- **프런트엔드**: React 18 + TypeScript + Vite, HashRouter, Zustand(persist), 손수 작성한 시맨틱 CSS.
- **이음새(어댑터)**: 인증·결제·eSIM 프로비저닝·푸시를 인터페이스로 분리 → 현재는 목업,
  환경변수로 **실제 백엔드(BFF) 전환** 가능.
- **번역 엔진**: 브라우저 온디바이스 Translation API 우선, 미지원 시 **MyMemory**(무료·무키) 폴백 + 캐시.
- **배포**: GitHub Actions로 GitHub Pages 자동 배포. 상대경로 + HashRouter라 **Figma Sites 등 iframe 임베드**에 그대로 적합.

## 11. 데모(MVP) 범위와 한계 — 확정 전 항목

- **가격**: 합리적 **예시 금액**(확정 단가 아님).
- **혜택**: 구체 할인율 대신 **"제공 예정"** 표기(제휴처별 상이).
- **결제·개통**: 목업(실제 PG·SM-DP+ 연동 미정, 이음새만 구현).
- **이미지**: 데모용 외부(Unsplash) 사진 — 운영 시 교체.
- **번역**: 무료 MyMemory(일일 한도 존재) — 운영 시 상용 엔진 전환 가능.
- **L.POINT 적립/혜택**: 실제 멤버십 연동 전 시연 수준.

## 12. 향후 로드맵(예시)

- 실제 PG 결제 + SM-DP+ eSIM 프로비저닝 연동.
- L.POINT 멤버십/적립 실연동, 혜택 쿠폰함·면세 픽업 예약.
- 라이프사이클 푸시(도착·소진·근처 혜택) 실발송.
- 후기 모더레이션, 상용 번역 엔진, 운영용 이미지/콘텐츠.
- zh/ja 잔여 라벨 완역.
